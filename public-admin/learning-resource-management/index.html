Governance Notice

Learning Resource Management is the authoritative administrative surface for registering and publishing learning-resource metadata within Agile AI University.

Protected licensed materials remain in Firebase Storage and are not exposed using permanent client-side download URLs. Student access is provided through governed identity, authorization, entitlement, release-policy resolution, and backend-brokered delivery.

Published resource versions are immutable. Corrections and replacements require a new version, while withdrawn resources remain available for governance and audit history.

ADR-019 governs how protected resources are delivered. ADR-020 governs when and to whom resources become available. The Admin Portal records release metadata, while the Learning Resource Resolver remains the authority for learner visibility.
