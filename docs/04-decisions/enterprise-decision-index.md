# Purpose

The Enterprise Architecture Decision Index provides the authoritative catalogue of Architecture Decision Records maintained within the Agile AI University repository.

It enables architects, developers, administrators, product owners, and governance reviewers to locate significant architectural decisions, understand their current status, and identify the systems, Domains, Platforms, and Enterprise Services affected by each decision.

The detailed decision context, alternatives, rationale, and consequences remain within the individual ADR documents.

---

# Objectives

This index exists to:

- Provide a single catalogue of architectural decisions
- Improve ADR discoverability
- Preserve decision traceability
- Identify superseded decisions
- Connect decisions with implementation
- Support architecture governance
- Prevent repeated architectural debates
- Preserve institutional knowledge

---

# Decision Authority

Architecture Decision Records are the authoritative source for significant architectural decisions.

An ADR records:

- Context
- Problem
- Decision
- Alternatives considered
- Rationale
- Consequences
- Governance implications
- Implementation impact
- Related documentation

This index summarizes ADRs.

It does not replace them.
