# Agile AI Specification

The Agile AI Specification defines the conceptual architecture and operational model for designing and operating AI-enabled organizational systems.

Maintained by the Agile AI Foundation.

## Specification Sections

Architecture
Defines the operating model of Agile AI systems.

Domain
Defines conceptual foundations and capability models.

Registries
Defines canonical identifiers used across the ecosystem.

Ecosystem
Defines institutional and governance structures.
